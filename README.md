# 股票交易记录系统

## 项目概述
这是一个基于Python的股票交易记录系统，使用Flask框架开发。系统支持港股和美股的交易记录管理，包括交易明细、费用计算、汇率管理、股票管理和盈亏统计等功能。

## 技术栈
- Web框架：Flask 3.0.2
- 前端框架：Bootstrap 5
- 数据库：MySQL
- ORM：SQLAlchemy 3.1.1
- 前端组件：
  - Font Awesome 5.15.4
  - Bootstrap Icons 1.7.2
  - Select2 4.1.0
  - Toastify（提示消息）
- 股票数据：yfinance 0.2.36

## 功能特性
- [x] 用户认证
  - 用户注册（用户名+密码）
  - 用户登录
  - 用户退出
  - 会话管理（7天有效期）

- [x] 股票管理
  - 支持港股和美股市场
  - 自动获取股票信息
  - 支持谷歌查询代码
  - 实时股价更新（每60秒自动刷新）
  - 股票名称管理
  - 批量价格查询
  - 股票代码自动完成
  - 股票删除保护（有交易记录的股票不可删除）

- [x] 股票交易记录管理
  - 支持港股和美股市场
  - 记录交易明细（买入/卖出）
  - 支持多笔成交记录
  - 费用明细记录
    - 经纪佣金
    - 交易征费
    - 印花税
    - 交易费
    - 存入证券费
  - 自动计算交易费用
  - 跟踪交易盈亏
  - 便捷输入功能
    - 交易日期默认为当天
    - 支持日期快速输入（如：20250103、0103、103、03）
    - 股票代码自动完成（基于历史记录）
    - 根据交易编号自动判断买卖类型（P开头为买入，S开头为卖出）
  - 键盘导航优化
    - 回车键导航顺序（固定）：
      1. 日期 -> 股票代码
      2. 股票代码 -> 交易编号
      3. 交易编号 -> 交易类型
      4. 交易类型 -> 第一个数量输入框
      5. 数量 -> 价格
      6. 价格 -> 经纪佣金（回车）或添加成交记录按钮（Tab）
      7. 经纪佣金 -> 交易征费
      8. 交易征费 -> 印花税
      9. 印花税 -> 交易费
      10. 交易费 -> 存入证券费
      11. 存入证券费 -> 保存并添加下一条（自动触发）
    - Tab键导航顺序（固定）：
      1. 经纪佣金 -> 保存按钮
      2. 交易征费 -> 印花税
      3. 印花税 -> 交易费
      4. 交易费 -> 存入证券费
      5. 存入证券费 -> 取消按钮
    - 特殊控件处理：
      - 价格输入框：
        - 回车键：跳转到经纪佣金
        - Tab键：跳转到添加成交记录按钮
      - 存入证券费：
        - 回车键：触发"保存并添加下一条"按钮
        - Tab键：跳转到取消按钮
    - 表单提交优化：
      - 禁用表单默认提交行为
      - 使用AJAX异步提交
      - 提交前进行表单验证
      - 根据不同按钮执行不同后续操作
  - 表单验证和错误提示

- [x] 汇率管理
  - 自动抓取美元汇率
  - 支持手动添加汇率记录
  - 汇率历史记录查询
  - 汇率数据编辑
  - 自动补全历史汇率
  - 汇率来源标记
    - 实时汇率
    - 历史汇率
    - 手动修改
  - 临时汇率自动更新

- [x] 统计分析
  - 市场维度统计
    - 买入总额
    - 卖出总额
    - 总费用
    - 净盈亏
    - 盈亏率
    - 实时市值
  - 股票维度统计
    - 当前持仓
    - 交易次数
    - 买入成本
    - 卖出收入
    - 已实现盈亏
    - 未实现盈亏
    - 盈亏率
    - 实时市值
  - 交易记录明细
    - 树形结构展示
    - 支持展开/收起
    - 持仓股票默认展开
  - 日期范围查询
  - 股票代码筛选
  - 实时价格更新（每60秒自动刷新）

## 开发环境配置
1. 克隆项目
```bash
git clone [项目地址]
cd stock
```

2. 创建虚拟环境
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
.\venv\Scripts\activate  # Windows
```

3. 安装依赖
```bash
pip install -r requirements.txt
```

4. 创建数据库
```sql
CREATE DATABASE Stock CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

5. 配置数据库连接
编辑 `config.py` 文件，修改数据库连接信息：
```python
SQLALCHEMY_DATABASE_URI = 'mysql+pymysql://用户名:密码@主机:端口/Stock'
```

6. 启动服务
```bash
python main.py
```

访问 http://localhost:9009 即可使用系统。

## 使用说明
1. 用户管理
   - 注册新账号：提供用户名和密码
   - 登录系统：使用已注册的账号
   - 退出登录：点击导航栏的"退出"按钮

2. 股票管理
   - 添加新股票
     - 选择市场（HK/USA）
     - 输入股票代码
     - 系统自动获取股票信息
     - 确认添加
   - 编辑股票信息
     - 修改股票名称
     - 更新谷歌查询代码
   - 查看实时价格
     - 系统每60秒自动更新所有股票价格
   - 删除股票（仅限无交易记录的股票）

3. 交易记录管理
   - 添加新的交易记录
     - 填写基本信息（日期、市场、股票代码等）
     - 添加成交明细（可添加多笔成交记录）
     - 填写费用明细
     - 使用键盘快速导航
       - Enter键：在输入框间快速切换
       - Tab键：跳转到特定按钮
   - 查看交易记录列表
     - 支持日期范围查询
     - 支持股票代码筛选
     - 显示详细的交易信息和费用明细
   - 编辑现有记录
   - 删除交易记录

4. 汇率管理
   - 查看汇率记录
   - 手动添加汇率
   - 自动抓取缺失汇率
   - 编辑已有汇率
   - 按日期范围查询
   - 查看汇率来源

5. 统计分析
   - 查看市场统计
     - 总投资金额
     - 当前市值
     - 盈亏情况
   - 查看股票统计
     - 持仓信息
     - 成本分析
     - 实时盈亏
   - 展开/收起明细
   - 按日期范围筛选
   - 按股票代码筛选
   - 实时更新市值

## 注意事项
- 这是一个开发版本，不建议在生产环境中直接使用
- 使用前请确保MySQL服务已启动
- 默认监听9009端口，如需修改，请在main.py中更改
- 股票数据来源于Yahoo Finance，可能存在延迟或不准确
- 建议定期备份数据库
